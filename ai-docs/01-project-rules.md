# Project Rules

Build a Flipkart-inspired ecommerce web app using:

- Next.js App Router
- TypeScript
- Tailwind CSS
- Prisma
- PostgreSQL
- Zustand

Rules:
- Clean folder structure
- Reusable components
- Small files
- Clear comments
- No giant components
- Use loading states
- Responsive design
- Keep code interview-ready
- Use TypeScript types properly
- Prefer server components where useful
